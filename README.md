# Docker Desktopの容量を開ける方法
### 未使用のDockerイメージを削除する場合は、
```
docker image prune -a
```
### 全てのDocker イメージを削除する
```bash
docker rmi $(docker images -q)
または
docker image rm $(docker image ls -aq)
```
```powershell
for /f %i in ('docker images -q') do docker rmi %i
```
